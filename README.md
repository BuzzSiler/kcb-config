#KeyCodeBox Alpha
raspberry-pi touchscreen lockbox appliance

## Dependencies
- >= QT5.3.2
- libxml2
- fprint https://www.freedesktop.org/wiki/Software/fprint/libfprint/
- g++

## Building
```
$ git clone https://github.com/petrichorsystems/keycodebox.git
$ cd keycodebox/alpha
$ qmake && make
```
